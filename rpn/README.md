# rpn
逆ポーランド記法に **似た** 記法を用いた暗号の復号化

## RUN
```sh
python3 rpn.py < crypt.txt
```

暗号
```
2 5 5 * - 7 1 / + 8 *
```

復号化
```
W
```


## 逆ポーランド記法に似た記法の変換

変換前
```
2 5 5 * - 7 1 / + 8 *
```

変換後
```
2 - 5 * 5 + 7 / 1 * 8
```

## 暗号変換

| 0 ~ 9 | 10 | 11 ~ 36 | 37 | 38 | 39 |
| :---: | :---: | :---: | :---: | :---: | :---: |
| 0 ~ 9 | ` ` | A ~ Z | . | , | \n |

変換前 (計算の答え)
```
33
```

変換後
```
W
```
